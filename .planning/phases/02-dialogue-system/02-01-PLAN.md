<objective>
Establish the two prerequisites that all Wave 1+ tasks require:
1. Register `dialogue_data` and `dialogue_manager` as Godot autoloads in `project.godot`

```gdscript
var npc_state: Dictionary = {}
```

**2. Save entry** — inside `save_to_slot()`, add before the final `cfg.save(...)` call
   (currently line 88). The new line goes after the existing `cfg.set_value("meta", ...)` line:

```gdscript
cfg.set_value("dialogue", "npc_state", var_to_str(npc_state))
```

**3. Load entry** — inside `load_from_slot()`, add after the `player_loaded_pos = Vector2(...)`
   block (currently ends at line 105), before `game_first_loading = false`:

```gdscript
var raw := cfg.get_value("dialogue", "npc_state", "{}")
npc_state = str_to_var(raw) if raw != "{}" else {}
if npc_state == null:
    npc_state = {}
```

The null guard on the third line defends against Pitfall 5 (str_to_var returning null on
certain inputs in Godot 4.x).

**4. Reset entry** — inside `reset_for_new_game()`, add after `transition_scene = false`
   and before `loaded_from_save = false` (currently lines 69-70):

```gdscript
npc_state = {}
```

No other changes. Do not modify function signatures or reorder existing lines.
